## 基于vite + vue3 + ts的后台管理项目

### 安装依赖
```
yarn
```

### 启动开发环境
```
yarn dev
```

### 构建生产环境
```
yarn build
```

### 生产环境预览
```
yarn preview
```

### eslint检查代码风格
```
yarn lint
```

### prettier格式化代码
```
yarn format
```

### 启动文档开发环境
```
yarn docs:dev
```

### 构建文档
```
yarn docs:build
```

### 预览构建文档
```
yarn docs:serve
```

### 需要安装的vscode插件
- Eslint
- Prettier
- Vetur
